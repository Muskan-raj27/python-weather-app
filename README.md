# python-weather-app# 🌦️ Weather App

A simple Weather App developed using **Python** and **Tkinter** that provides real-time weather information for any city. The application retrieves weather data using a weather API and displays details such as temperature, humidity, weather conditions, and wind speed through a user-friendly graphical interface.

# 📌 Project Description

The Weather App enables users to enter the name of any city and instantly view its current weather conditions. The application fetches live weather information from a weather API and displays details such as temperature, humidity, weather description, wind speed, and location. It is designed to be simple, fast, and user-friendly.

# ▶️ How to Run

1. Install Python 3.
2. Install the required library:

```bash
pip install requests
```

3. Add your Weather API key in the code.
4. Run the application:

```bash
python weather_app.py
```

# 🚀 How to Use

1. Launch the application.
2. Enter the city name.
3. Click the **Get Weather** button.
4. View the current weather details.
